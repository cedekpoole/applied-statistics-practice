# Applied Statistics Practice

A small Python project for practising applied statistics and statistical modelling using real-world datasets.

## Aim

The aim of this project is to build clear understanding of core statistical workflows, including:

- data inspection
- missing values
- descriptive statistics
- statistical tests
- confidence intervals
- linear regression
- logistic regression
- model evaluation
- interpretation and limitations

## Datasets

This project uses:

- Palmer Penguins
- UCI Bike Sharing
- UCI Bank Marketing
